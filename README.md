## О проекте


Этот проект был создан с помощью gemini и был доработан claude, да, бот нормально работает можно делать всё с ним давать пропмт и тд

📋 Список всех команд
Основные

Просто напиши сообщение — бот ответит
Отправь фото с подписью — бот проанализирует
1. /ask <вопрос> — задать вопрос явно

Управление историей

1. /clear — очистить историю диалога
2. /history — показать статистику и последние сообщения

Настройки
1. /model <название> — сменить модель Gemini
2. /models — список доступных моделей с кнопками
3. /currentmodel — показать текущую модель
4. /setprompt <текст> — установить системный промпт
5. /resetprompt — сбросить промпт на стандартный
6. /myprompt — показать текущий промпт

Информация

1. /stats — твоя статистика (запросы, история, модель)
2. /help — список всех команд
3. /start — приветствие

Конечно же бот почти сырой, там очень мало функций. Но я постараюсь добавить всё нужное туда.

### Установка

_Ниже предоставлены комманды для получения скрипта запуска gemini для телеграмма_

1. Получите API Key at [AI-Studio](https://aistudio.google.com/app/apikey)
2. Колнируйте репозиторий
   ```console
   git clone https://github.com/AimRite2YT/gemini-telegram-bot.git
   ```
3. Введите свой API ключ в `config.py`
   ```api_key
   GEMINI_API_KEY = 'ENTER YOUR API';
   ```
4. Создайте бота через @BotFather, скопируйте токен и вставьте его сюда:
   ```telegram_token
   TELEGRAM_BOT_TOKEN = 'ENTER YOUR TOKEN'
   ```
5. Запустите скрипт
   ```bash
   python telegram_bot.py
   ```
